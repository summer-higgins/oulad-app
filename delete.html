Delete Data Description

In this webpage, users are allowed to delete records from the OULAD database. Each section on the right corresponds to a specific table. Enter the required key fields carefully.

Delete Instructions

To delete data correctly, use the identifying fields for the matching table. If the key values do not exactly match an existing record, nothing will be deleted.

  • Student Information

    Use Student ID, Code Module, and Code Presentation.

  • Student Registration

    Use Student ID, Code Module, and Code Presentation.

  • Student Assessment

    Use Student ID and Assessment ID.

  • Total Clicks

    Use Student ID, Code Module, and Code Presentation.

  • Weekly Clicks

    Use Student ID, Code Module, Code Presentation, and Week.

  • Activity Type Clicks

    Use Student ID, Code Module, Code Presentation, and Activity Type.
