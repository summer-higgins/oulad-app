Welcome to the OULAD Student Engagement Analytics Dashboard

This dashboard provides insights into student engagement based on the OULAD dataset. You can add, delete, and update student data, as well as view summaries and documentation about the dataset.

Use the navigation links above to explore the different functionalities of the dashboard.

Project Overview

The purpose of this application is to provide a simple way for users to store, query, and analyze student performance and engagement data from online courses. The app helps users understand how factors such as student background, registration behavior, assessment performance, and VLE activity may influence academic outcomes.
